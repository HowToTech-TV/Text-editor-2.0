# Text Editor 2.0

- A simple tool for opening text or coding files, written in python.

# Features in Ver. 2.1

- Run Python Files (Py or Pyw)
- With font changer support
- Extensions
- Open and Save Files
- Fullscreen
- Dark Theme Support
